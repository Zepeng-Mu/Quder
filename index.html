Qubit Quantification Protocol

Well Preparation
  • In each well: 98µl buffer solution + 2µl sample (or standard)
  • If using 384-well plate: 39µl + 1µl sample (or standard)
  • Buffer Solution: use 1x HS DNA buffer
Standards

Serial dilution to make 8 standards in a column of a 96w plate or 8-strips.

  • 8 standards: 10, 5, 2.5, 1.25, 0.625, 0.3125, 0.156, 0
  • Start with 25µl of S1 (0) in each well except the first one.
  • Add 50µl of S2 (10) in first well.
  • Take 25µl from the first well and transfer to the next well. Mix well! Transfer 25µl from the second well to the third well, etc. Leave tube 8 with only the S1.
  • SAVE until runs out or R² is not less than 98.
Plating & Reading
  • Pipette first the buffer solution to each well you will use.
  • Recommendation: I usually use a 384w plate and put samples into every other column using a multichannel pipette. I use the lower rows N, O & P for the standards, and pipette those in triplicate.
  • Vortex plates VERY VERY well and read in Synergy H1 Hybrid Reader, using Qubit_384w_protocol.
